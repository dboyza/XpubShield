# XpubShield 0.1.0 Windows Builds

Use one of these files to run XpubShield without installing Node, npm, Rust, or the source repo.

- `XpubShield_0.1.0_x64-setup.exe`: setup wizard, recommended for most users.
- `XpubShield_0.1.0_x64_en-US.msi`: MSI installer for managed Windows installs.
- `xpubshield.exe`: direct executable from the release build.

These builds are unsigned, so Windows SmartScreen may show a warning.
